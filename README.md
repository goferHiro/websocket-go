# Websocket-Go

A minimalistic websocket (echo-server) implemented in go

## Prerequisites

1. Websocket client
2. https://www.piesocket.com/websocket-tester


## Usage

```bash
git clone https://github.com/goferHiro/websocket-go
cd websocket-go
go run .
```
